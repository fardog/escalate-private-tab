Browser extension for reopening a private tab as a regular tab, and vice versa. This extension was forked from cpeterso/open-page-in-private-window/ as it provided a reasonable base to build from. It has been updated to Manifest V3 to support multiple browsers in theory, but has only been tested on Firefox. Note: You must allow this extension in Private Windows. You may follow the instructions here to enable this permission: https://support.mozilla.org/kb/extensions-private-browsing
